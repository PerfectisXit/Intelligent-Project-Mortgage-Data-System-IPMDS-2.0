# IPMDS v0.1.0

Initial public baseline release for **Intelligent Project Mortgage Data System (IPMDS) 2.0**.

## Highlights
- End-to-end ledger workflow (import → review → commit/rollback)
- AI copilot with structured JSON parsing and fallback routing
- OCR document extraction + unit linking (auto / manual confirm)
- PostgreSQL-first transactional model with import field-level audit trail
- RBAC access controls with auditor-sensitive-field masking
- CI + local verification script for stable delivery

## Included Modules
- `frontend/`: React + Ant Design workbench
- `backend/`: Node.js TypeScript APIs
- `python-service/`: FastAPI diff/OCR services
- `database/migrations/`: PostgreSQL schema and audit migrations

## API Summary
- `POST /api/v1/copilot/interpret`
- `POST /api/v1/copilot/commit`
- `POST /api/v1/imports/excel`
- `GET /api/v1/imports/{id}/diff`
- `GET /api/v1/imports/{id}/audits`
- `POST /api/v1/imports/{id}/commit`
- `POST /api/v1/imports/{id}/rollback`
- `POST /api/v1/files/ocr-link`
- `POST /api/v1/files/{id}/confirm-link`

## Notes
- Default local mode can run with `MOCK_MODE=true`.
- Production should run with `MOCK_MODE=false` and real PostgreSQL.
